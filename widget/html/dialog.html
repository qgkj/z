请先看以下步骤和说明进行测试
  • 1.安卓版本必须不小于4.5；
  • 2.默认推流到服务器为阿里云：1核CPU / 1G 内存 / 15G流量包 / 系统盘 20G Linux，修改推流地址后，播放端也要相应的修改该地址，一般测试不需进行此项；
  • 3.第一步：填写推流地址（请先使用默认地址），设置分辨率（240 / 360 / 480 / 540 / 720 / 1080），设置帧率；
  • 4.第二步：点击配置直播流（默认使用后置摄像头）；
  • 5.第三步：点击开始推流就进行直播了；
  • 6.测试观看地址为：地址被我去掉了，因为服务器流量原因，如需进行测试和交流请左上角联系我，只支持PC端浏览器观看，手机端观看正在加紧研究中；
  • 7.我们当前测试用的服务器案例只有15G流量，在您测试完后请点击“停止推流”以节省流量；
  • 8.此demo为初版，3月7日第一次实现了完整的直播，还需优化。
